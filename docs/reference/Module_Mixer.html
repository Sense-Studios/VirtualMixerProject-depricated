Class: Mixer

Module#Mixer(renderer:GlRenderer, options:Object)

new Mixer(renderer:GlRenderer, options:Object)

A mixer mixes two sources together.

It can crossfade the sources with different MixModes and BlendModes
requires source1 and source2 in options both with a Source (or another Module like a Mixer)
